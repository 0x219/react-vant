## 特性

- 30+ 高质量组件，覆盖移动端基本场景
- 支持按需引入
- 支持主题定制
- 支持 TypeScript

## 📦 Init App

npm

```bash
npm install
```

yarn

```bash
yarn
```

## 🔨 使用

### 启动文档

```bash
yarn start
```

### 打包文档

```bash
yarn docs:build
```

### 发布文档

```bash
yarn docs:deploy
```

### 打包组件

```bash
yarn build
```

### 测试组件

```bash
yarn test
```

### 发布 npm 包

```bash
yarn release
```
