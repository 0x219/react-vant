### 介绍

旨于参照 [Vant](https://vant-contrib.gitee.io/vant) 打造 React 框架下移动端的优秀组件库。

### 特性

- 30+ 高质量组件，覆盖移动端基本场景
- 支持按需引入
- 支持主题定制
- 支持 TypeScript

### 快速上手

请参考[快速上手](#/zh-CN/quickstart)章节。

### 浏览器支持

现代浏览器以及 Android 4.0+, iOS 8.0+。
